# EMDR-Art

A responsive website developed for EMDR therapist and artist Lee, designed to provide information about EMDR-based art therapy, promote training courses, share publications, and support community engagement.

The site was built using HTML, CSS, and JavaScript and focuses on accessibility, clear navigation, and a professional user experience for prospective clients, therapists, and course participants.

---

## Features

### Homepage

* Introduction to Lee's work and EMDR-Art practice
* Clear navigation to services, publications, and resources
* Responsive design for desktop and mobile devices

### Course Booking

* Dedicated booking page for training courses
* Structured booking form for collecting attendee information
* User-friendly registration experience

### Publications & Resources

* Access to publications and professional resources
* Central location for research and educational materials

### Blog Integration

* Links to existing blog content and external resources
* Enables ongoing sharing of insights and updates

### Mailing List

* Newsletter sign-up functionality
* Supports audience engagement and future communications
